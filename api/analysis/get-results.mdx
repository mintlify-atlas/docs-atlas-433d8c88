## Overview

This endpoint retrieves the legacy model analysis results for a specific post. It returns the analysis only if it was previously computed using the old model endpoint.

<Note>
This endpoint only retrieves existing results. To generate new analysis, use the [Analyze with Old Model](/api/analysis/old-model) endpoint.
</Note>

## Authentication

This endpoint requires authentication via JWT token.

## Path Parameters

<ParamField path="postId" type="string" required>
  The MongoDB ObjectId of the post whose old model analysis you want to retrieve
</ParamField>

## Response

Returns an `OldModelAnalysis` object containing the legacy model's analysis results.

<Expandable title="Response Schema">
  <ResponseField name="_id" type="string">
    Analysis document ID
  </ResponseField>

  <ResponseField name="post_id" type="string">
    Reference to the analyzed post
  </ResponseField>

  <ResponseField name="is_fake" type="boolean">
    Overall determination if the video is fake according to the old model
  </ResponseField>

  <ResponseField name="confidence" type="number">
    Overall confidence score from the old model
  </ResponseField>

  <ResponseField name="frames_analysis" type="array">
    Array of per-frame analysis results
    
    <Expandable title="frame object">
      <ResponseField name="frame" type="string">
        Frame filename
      </ResponseField>

      <ResponseField name="confidence" type="number">
        Confidence score for this frame (0-1)
      </ResponseField>

      <ResponseField name="is_fake" type="boolean">
        Whether this frame is detected as fake
      </ResponseField>

      <ResponseField name="frame_path" type="string">
        Path to the frame image file
      </ResponseField>
    </Expandable>
  </ResponseField>

  <ResponseField name="summary" type="object">
    Analysis summary statistics
    
    <Expandable title="summary properties">
      <ResponseField name="status" type="string">
        Overall verdict: "FAKE" or "REAL"
      </ResponseField>

      <ResponseField name="confidence_percentage" type="number">
        Confidence as percentage (0-100), calculated as (fake_frames / total_frames) * 100
      </ResponseField>

      <ResponseField name="total_frames" type="number">
        Total number of frames analyzed
      </ResponseField>

      <ResponseField name="real_frames" type="number">
        Count of frames classified as real
      </ResponseField>

      <ResponseField name="fake_frames" type="number">
        Count of frames classified as fake
      </ResponseField>
    </Expandable>
  </ResponseField>

  <ResponseField name="createdAt" type="string">
    Timestamp when analysis was created
  </ResponseField>

  <ResponseField name="updatedAt" type="string">
    Timestamp of last update
  </ResponseField>
</Expandable>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET https://api.example.com/api/posts/old-model-analysis/507f1f77bcf86cd799439011 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'https://api.example.com/api/posts/old-model-analysis/507f1f77bcf86cd799439011',
  {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);

const analysis = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/api/posts/old-model-analysis/507f1f77bcf86cd799439011',
    headers={
        'Authorization': f'Bearer {token}'
    }
)

analysis = response.json()
```
</CodeGroup>

## Error Responses

<ResponseField name="404 Not Found" type="object">
  No old model analysis exists for the specified post
  
  ```json
  {
    "message": "Analysis not found"
  }
  ```
</ResponseField>

<ResponseField name="500 Internal Server Error" type="object">
  Database or server error while fetching analysis
  
  ```json
  {
    "message": "Error message details"
  }
  ```
</ResponseField>

<ResponseField name="401 Unauthorized" type="object">
  Missing or invalid authentication token
  
  ```json
  {
    "message": "Unauthorized"
  }
  ```
</ResponseField>

## Use Cases

- Retrieve cached old model results for comparison
- Display historical analysis data
- Check if old model analysis exists before triggering new analysis
- Build comparison dashboards between model versions

<Info>
**Performance Note**: This is a read-only operation that doesn't trigger any analysis. It's fast and efficient for retrieving previously computed results.
</Info>
